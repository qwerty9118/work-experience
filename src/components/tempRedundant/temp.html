filing system idea2

widgets: [
{ category: acategoryname , name: "exampleWidget", picture: "an image", contents: "???" }
{ category: acategoryname.asubcategoryname.etc , name: "exampleWidget", picture: "an image", contents: "???" }
]

Widget variables:

  • name - title/name of widget.
  • picture - an image to go next to the title, if there is one.
  • contents - could be anything from a clock, to a chart, to a tweet feed, to a timetable, to misc. reminders.
  • [optional] category - the (simulated) location of the widget
  • [optional] id - which widget it is
